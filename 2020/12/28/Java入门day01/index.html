java中分基本数据类型、引用数据类型（类、数组、接口、接口对应的抽象类等）

  • 重载问题

一个类中可以有多个方法具有相同的名称，但这些方法的参数名必不相同

可以是 1、参数个数不同

​ 2、参数个数相同，但参数类型不同

  • Private

让类里面的属性不被外界访问，在属性面前加private

若要访问，可通过添加一个函数方法，通过方法访问私有属性

也可通过set方法给属性赋值

快速设置多个已声明属性的get或set方法：

1、右键->Generate->Getter/Setter->选中需要设置该方法的属性->OK

2、Alt+insert->Generate->Getter/Setter->选中需要设置该方法的属性->OK

  • 构造方法

1、无参构造方法（默认，不写也会有）
2、有参构造方法

通过传参可在new时给声明的类赋予相对应的属性值

  • 堆、栈、方法区

Java程序运行时，数据会分区存放，JavaStack（Java栈）、 heap（堆）、method（方法区）

栈内存中存放基本数据类型的数据，和对象的引用（堆内存地址）

堆内存存放类的对象，所有的类对象都是通过new方法创建，创建后，在栈中会创建类对象的引用（内存地址）

方法区，又叫静态区，存放所有的类（class）、静态变量、静态方法、常量和成员方法

  • 垃圾回收机制

与C++不同的是，不用每次都手动回收清除，Java是自动回收

  • 重载和重写的区别

重写（override）

override是重写（覆盖）了一个方法，以实现不同的功能。一般是用于子类在继承父类时，重写（重新实现）父类中的方法

重载（overload）

一般是用于在一个类内实现若干重载的方法，这些方法的名称相同而参数形式不同

【类不支持多继承】

  • 多态

多态是同一个行为具有多个不同表现形式或形态的能力。

多态就是同一个接口，使用不同的实例而执行不同操作

多态存在的三个必要条件

  • 继承
  • 重写
  • 父类引用指向子类对象：Parent p = new Child();

  • 抽象类

抽象方法只能存在抽象类中，抽象类中可以拥有普通方法

关键字abstract

抽象类只能用来被继承

当子类继承了父类，且父类为抽象方法时，子类必须重写方法

  • 接口

通过interface关键字声明

抽象方法可以不写方法体public abstract

没有方法体的方法一定是抽象方法

在接口中不允许有变量存在

接口是一个特殊的抽象类，也是用来被子类继承的，有一个特殊的关键字implements用来实现接口
